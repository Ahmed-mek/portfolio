# Web Development & Interactive Lighting Simulation

**Category:** PC Software  
**Status:** ✅ Completed (2021 – 2022)  
**Organization:** Zenopix Technology, Konya, Turkey

---

## Overview

A dual-focus web development project encompassing an **interactive JavaScript/CSS lighting simulation** for customer-facing product visualization, and the design and launch of **four localized multilingual corporate websites** with full SEO optimization for improved search engine discoverability.

Though not a hardware project, this work demonstrates full-stack web engineering skills — interactive DOM programming, CSS animation, multilingual content architecture, and SEO — directly applicable to embedded product dashboards, data visualization interfaces, and corporate web presence.

---

## Key Contributions

### Interactive Lighting Simulation
- Developed a real-time **interactive JavaScript/CSS lighting simulation** demonstrating environmental lighting effects for customer-facing product visualization.
- Implemented dynamic light sources, ray-casting-inspired shadow effects, and real-time color mixing using Canvas API and CSS custom properties.
- Optimized the simulation for smooth 60 FPS performance across desktop browsers using `requestAnimationFrame` and efficient DOM manipulation patterns.

### Multilingual Corporate Websites
- Designed, developed, and launched **four localized multilingual corporate websites**, each serving a different regional market with full language localization.
- Implemented responsive layouts (mobile-first design) compatible with all screen sizes using HTML5, CSS3, and vanilla JavaScript.
- Managed multilingual content architecture: language switcher, per-locale routing, and dynamic content loading.

### SEO Optimization
- Implemented comprehensive **SEO optimizations** across all websites:
  - Proper semantic HTML5 structure (h1/h2 hierarchy, article/section/nav tags).
  - Unique meta titles and meta descriptions per page.
  - Open Graph and Twitter card metadata for social sharing.
  - XML sitemap generation and robots.txt configuration.
  - Performance optimization: image compression, lazy loading, minified CSS/JS.
- Achieved measurable improvements in search engine indexing and organic discoverability.
